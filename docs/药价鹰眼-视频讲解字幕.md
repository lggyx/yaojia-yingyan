# 药价鹰眼 — 演示视频讲解字幕（5 分钟版）

> 用于语音生成，8 段约 4.5 分钟，中文语速约 250 字/分钟。

---

## 第一段：片头（约 20 秒）

```
【画面：深绿色品牌背景 + 药价鹰眼 Logo + 副标题】
```

医保基金是老百姓的救命钱。药品价格虚高、地区差价异常、集采药被高价替代——这些都在侵蚀基金安全。

药价鹰眼是一套面向医保价格治理的智能监测系统，跑通了一条从发现异常到闭环处置的完整链路。

---

## 第二段：工作台总览（约 35 秒）

```
【画面：今日工作台，从导航到 KPI 卡片，再到 AI 简报和任务列表】
```

打开系统，左侧五个导航入口覆盖了监管处置全流程。

顶部 KPI 卡片一目了然：监测记录数、异常候选、高风险数量、闭环率。往下是 AI 简报，自动分析当前状态，告诉你今天优先关注什么。再往下是任务列表，系统按紧急程度自动生成，点击"开始处理第一项"直接跳转。

---

## 第三段：异常研判全流程（约 60 秒）

```
【画面：异常研判页，依次展示命中规则 → AI 取证 → 红队校验 → 研判报告 → 创建工单，节奏紧凑】
```

切换到"异常研判"页面，左边价格表标注了风险等级，点击一条进入研判步骤流。

第一步命中规则：采购价超中标价 1.5 倍、历史环比涨幅超 30%、地区偏差超 50%——规则引擎是纯确定性数学计算，不依赖 AI，速度快、可审计。

第二步 AI 取证：智能体自动调用工具链，查中标价、历史价、区域均价、替代药，然后由 LLM 用专业中文解释"为什么异常"。每一步调用都记录在案，可追溯。

第三步红队校验：系统主动质疑——规格剂型是否不可比？是否政策调价？是否特殊采购？逐条检验后计算置信度。多数反驳失败则确认异常，解释成立则降级或排除。这是对抗校验的核心——AI 自己跟自己辩论，降低误判。

第四步生成研判报告，汇总取证和校验结果，给出处置建议。确认异常后，点击创建工单，自动进入看板。

---

## 第四段：处置工单与复核闭环（约 45 秒）

```
【画面：四列看板 + 推进工单 + 复核 + 自动闭环/退回处置】
```

切换到"处置工单"，四列看板：待处置、处置中、已整改、已闭环。点击推进按钮，工单按状态机流转。右侧详情面板展示当前阶段和处置记录。

整改完成后点击 AI 复核，系统重新读取最新价格，判断是否回到阈值内。复核通过则自动闭环，工单从"已整改"移到"已闭环"。如果价格仍然偏离，提供"退回处置"按钮，把工单退回重新整改。闭环不是走形式，而是真正整改到位。

---

## 第五段：数据与规则（约 30 秒）

```
【画面：数据与规则三 Tab，快速切换展示】
```

"数据与规则"页面提供三个 Tab。价格记录 Tab 支持增删改查，有关联异常时拒绝删除，防止数据不一致。规则配置 Tab 可以调整四个检测阈值，保存后重检测，引擎用新参数跑一遍。标准化工具 Tab 输入原始药品描述，自动识别医保编码、通用名、规格、剂型，给出置信度。

---

## 第六段：AI 助手（约 25 秒）

```
【画面：右侧展开 CopilotSidebar，输入"今天先做什么"，展示回复和跳转】
```

每个页面右侧都有 AI 助手，点击展开即可对话。问"今天先做什么？"，助手基于当前数据给出优先级建议，附带可点击的建议操作和快速跳转。点击后自动跳转到对应页面并选中目标，侧栏关闭，马上开始工作。智能调度不是替代判断，而是帮人更快找到最需要关注的事。

---

## 第七段：架构总结（约 20 秒）

```
【画面：三层架构示意图：规则引擎 → 智能体 → 闭环工单】
```

系统三层架构：第一层规则引擎，确定性多维比价，稳定检出异常；第二层智能体，取证工具链加 LLM 解释加红队校验，把异常讲清楚、把误判降下来；第三层闭环工单，看板加状态机加复核，每条异常都有人跟、有人改、有人验。

---

```

```

药价鹰眼——规则可复现，证据可追溯，结论可反驳，处置可闭环。谢谢观看。
